# Educational Keylogger Project

⚠️ **Disclaimer**: This project is for **educational purposes only**.  
It demonstrates Python socket programming, client-server communication, and keystroke logging in a controlled lab environment.  
Do not use this project on machines without explicit permission.

## Features
- Python client-server architecture
- Real-time keystroke capture and transfer
- Cross-platform support (Linux & Windows)
- Error handling for special keys (Shift, Ctrl, etc.)

## Skills Demonstrated
- Python socket programming
- Multithreading
- Cross-platform debugging
- Cybersecurity lab practices
